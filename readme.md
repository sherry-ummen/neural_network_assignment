# Usage 

## pipenv install
## pipenv run python main.py

```
usage: main.py [-h] [-l] [-c] [-v] [-p] [-f FILE]

Options:

optional arguments:
  -h, --help            show this help message and exit
  -l, --lc              Plot Violin chart (Distribution between X and Avg user count)
  -c, --cor             Plot Correlation between Day and Avg user count
  -v, --verbose         Train model with Verbose
  -p, --predict         Print the prediction
  -f FILE, --file FILE  csv file path
```
